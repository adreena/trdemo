About Me


My name is Kimia Hassanzadeh, I'm currently pursing PhD at University of Alberta in Software Engineering and Intelligent Systems.

I'm interested in Web Intelligence, Unstructured Data querying and smart interactive user-interfaces


During my MSc. project I developed a software to convert texts into RDF graph , T2R. Graph consists of entities as vertices, and relations among entities as edges. Relations are either grammatical dependencies or semantic roles assigned to phrases of a sentence

Semantic Web is a smart and self-describing way of representing data all across the web. This style helps query engines to follow links and mine information automatically. My research was mostly about knowledge extraction and converting plain texts into machine-readable data, covering both syntactic and semantic aspects of texts. The project goal was changing every detail into triples and building a graph-data representation.

Code is mainly written in python , because of its powerful Natural Language Processing toolkit.

This system can be furthered used/studied in Event Detection , Query engines and Text summarization systems.
